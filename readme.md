# plate-game CLI

A game pototype about doing math with license plates.

## Setup

```
yarn install
yarn link
plate-game
```

## Publishing to NPM

To package your CLI up for NPM, do this:

```shell
$ npm login
$ npm whoami
$ npm lint
$ npm test
(if typescript, run `npm run build` here)
$ npm publish
```
